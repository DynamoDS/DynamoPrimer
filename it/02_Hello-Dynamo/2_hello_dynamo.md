

# CIAO DYNAMO!

Essenzialmente, Dynamo è una piattaforma di programmazione visiva ed è uno strumento di progettazione flessibile ed espandibile. Poiché può funzionare come applicazione indipendente o come componente aggiuntivo di altro software di progettazione, è utilizzabile per sviluppare un'ampia gamma di workflow creativi. Procedere all'installazione di Dynamo e iniziare esaminando le funzionalità principali dell'interfaccia.
